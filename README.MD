easy online python chat
Libraries used: socket, threading, requests, os, time;


To use it install the requirements folder. Open a shell in the folder directory and make sure you have python3 and pip installed, then write the command pip install -r requirements.txt. After installing the requirements you will need to run server.py and connect with the clients to it by using your server ip. You will also need to open port 8081 for it to work.
